# Calculator

A simple web-based calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/)
- Clear (C) button to reset the current entry
- Backspace (BS) button to delete the last digit
- Division by zero protection with error message
- Clean and intuitive user interface

## Files

- `index.html` - Calculator structure and layout
- `calculator.js` - Calculator logic and operations
- `style.css` - Calculator styling

## How to Use

1. Open `index.html` in a web browser, or you may open https://minhaj1403.github.io/calculator/ to inspect my implementation
2. Click the number buttons to enter values
3. Click an operator (+, -, *, /) to perform calculations
4. Press = to see the result
5. Use C to clear or BS to backspace
